# utils Subsystem

## Purpose
The `utils` subsystem houses common helper functions, file-handling routines, and data format utilities used across multiple SwitchBoard subsystems.

## Responsibilities
- Provide general utility functions (non-domain specific) to reduce code duplication.

## Public APIs
- No major public APIs for Phase 0 (internal helpers).

## Future Work
- Add repository directory traversal helpers.
- Add system hardware monitoring adapters (VRAM/RAM system calls).
